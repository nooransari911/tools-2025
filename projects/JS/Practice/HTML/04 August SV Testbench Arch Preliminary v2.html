Generator

Generates random stimulus transactions based on constraints.

Driver

Receives transactions and drives them onto the physical interface signals.

Monitors

Passively capture signal activity on the interface and convert it back into transactions.

Reference Model

A behavioral model that calculates the expected "golden" result for a given input.

Scoreboard

Compares expected results from the Reference Model with actual results from the Monitors.

DUT & Interface

The Design Under Test and the Virtual Interface that provides access to it.
